Basic actions

Basic actions are the ones that modifies basic attributes like:

  • position

    • MoveBy
    • MoveTo
    • JumpBy
    • JumpTo
    • Bezier
    • Place

  • scale

    • ScaleBy
    • ScaleTo

  • rotation

    • RotateBy
    • RotateTo

  • visible

    • Show
    • Hide
    • Blink
    • ToggleVisibility

  • opacity

    • FadeIn
    • FadeOut
    • FadeTo
